# Md Rafatullah Portfolio

Modern React, Tailwind CSS, and Framer Motion portfolio for Md Rafatullah.

## Local Development

```powershell
npm.cmd install
npm.cmd run dev -- --port 5173
```

Open `http://localhost:5173`.

## Build

```powershell
npm.cmd run build
```

## GitHub Pages

This repository includes `.github/workflows/deploy.yml`. After pushing to a GitHub repository, enable GitHub Pages from:

Settings > Pages > Build and deployment > Source > GitHub Actions

The site will deploy automatically after pushes to `main`.
